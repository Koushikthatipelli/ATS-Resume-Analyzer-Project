# ATS Resume Analyzer

A modern ATS (Applicant Tracking System) Resume Analyzer built using React and Vite.

## Features

* ATS Match Score Calculation
* Resume Skills Analysis
* Job Description Comparison
* Missing Skills Detection
* Personalized Recommendations
* GitHub Dark Theme UI
* Responsive Design

## Tech Stack

* React
* Vite
* CSS
* React Icons

## How It Works

1. Enter your resume skills.
2. Paste the job description.
3. Click **Analyze Resume**.
4. View:

   * ATS Score
   * Matched Skills
   * Missing Skills
   * Recommendations
